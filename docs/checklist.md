# Checklist de Testes

## Pré-Deploy

### Testes E2E
- [ ] Fluxos críticos testados
- [ ] Testes executando no ambiente de staging
- [ ] Todos os testes passando
- [ ] Screenshots/evidências em caso de falha

### Testes de API
- [ ] Todos os endpoints críticos testados
- [ ] Validações de schema implementadas
- [ ] Testes de erro/edge cases incluídos
- [ ] Autenticação/autorização validada

### Testes de Performance
- [ ] Testes de carga executados
- [ ] Métricas dentro dos SLAs
- [ ] Análise de gargalos realizada
- [ ] Relatório gerado e revisado

### Testes de Segurança
- [ ] Varredura de segurança executada
- [ ] Vulnerabilidades críticas/altas tratadas
- [ ] Revisão de dependências realizada

## Pós-Deploy

### Validação Rápida
- [ ] Smoke tests executados
- [ ] Funcionalidades críticas validadas
- [ ] Monitoramento ativado
- [ ] Logs verificados

## Regressão

### Testes Funcionais
- [ ] Suite de regressão completa
- [ ] Testes de integração validados
- [ ] Cross-browser (quando aplicável)

### Validações Específicas
- [ ] Compatibilidade de navegadores
- [ ] Responsividade (mobile/desktop)
- [ ] Acessibilidade básica

## Manutenção

### Revisão Periódica
- [ ] Testes obsoletos removidos
- [ ] Novos testes adicionados conforme features
- [ ] Performance dos testes monitorada
- [ ] Documentação atualizada
